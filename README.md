# 2ite-web-halusk--ondrej

## Stručný popis webu
Jednostránkový osobní web o Ondřeji Haluškovi. Web slouží jako prezentace osoby, jejího profilu, přístupu a kontaktu. Obsahuje úvodní hero sekci se sliderem, navigaci s anchor odkazy a responzivní rozložení pro mobilní zařízení.

## Použité technologie
- HTML5  
- CSS3 (Flexbox, Media Queries, responzivní design)  
- JavaScript (práce s DOM, slider, přepínání jazyka, animace při scrollu)

## Návod na spuštění
Projekt nevyžaduje žádný build.
- Po naklonování repozitáře stačí otevřít soubor `index.html` v internetovém prohlížeči.

## Prvky, na které jsem nejvíce hrdý
- Hero slider přes celou obrazovku (šipky, tečky, automatické přepínání)
- Responzivní horní navigace přizpůsobená mobilním zařízením
- Přepínač jazyka CZ / EN bez nutnosti obnovy stránky
- Animované zobrazování sekcí při scrollování
